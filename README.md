# Panamerican Taekwondo Academy Website

A professional, dynamic landing page for the Panamerican Taekwondo Academy.

## Features

- Responsive design with glassmorphism effects
- Smooth scrolling and animations
- Interactive button effects with 3D transforms
- Particle background effects
- Mobile-friendly navigation

## Installation

To install the project dependencies, run:

```bash
npm install
```

## Development

To run the website locally with a development server:

```bash
npm start
```

This will start a local HTTP server on port 8080. Open your browser and navigate to:
```
http://localhost:8080
```

Alternatively, to start the server and automatically open it in your browser:

```bash
npm run dev
```
